# SnippetVault 🚀

A self-deleting code sharing platform built with Go.

## 🎯 Goal
Share code snippets that automatically expire after a set time.

## 🛠️ Tech Stack
- **Backend:** Go (net/http)
- **Frontend:** Go Templ + HTMX + CSS
- **Database:** PostgreSQL
- **Infrastructure:** Docker, Makefile

## 🏗️ Project Structure
- `cmd/`: Application entry point
- `internal/`: Private business logic
- `ui/`: Frontend templates and static assets
